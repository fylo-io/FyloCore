## üìã System Requirements

### Minimum Requirements
- **Node.js**: 18.x or later
- **RAM**: 4GB
- **Storage**: 20GB
- **Database**: PostgreSQL 14+ (via Supabase)

### Production Requirements
- **CPU**: 4+ cores
- **RAM**: 8GB+
- **Storage**: 100GB+ SSD
- **Load Balancer**: Nginx or equivalent
- **Container Runtime**: Docker 20.10+
